⚡ Features

Unity brings powerful automation, moderation, and engagement tools to your Discord community. Here’s what makes it stand out:

  • 🔧 Easy setup for logs, welcome messages, tickets, and more.
  • 🛡️ Robust moderation tools for a safer server.
  • 🏆 Leveling and XP rewards to engage your members.
  • 📢 Reaction roles and role counters for community organization.
  • 🤖 Background automation for smooth server management.

🏆 Leveling System

Keep your community active with Unity’s leveling system:

  • Leaderboard: Public command showing the top 10 members.
  • Level Progress: Personal progress tracking at any time.
  • DM Notifications: Get notified when you level up or down.

📈 Level Progression

  • Newcomer: 0 messages
  • Active Participant: 10 messages
  • Contributor: 25 messages
  • Engaged Member: 50 messages
  • Dedicated Member: 100 messages
  • Community Advocate: 200 messages
  • Collaborator: 350 messages
  • Team Player: 500 messages
  • Leader-in-Training: 750 messages
  • Community Leader: 1000 messages
  • Mentor: 1500 messages
  • Trusted Voice: 2000 messages
  • Influencer: 3000 messages
  • Visionary: 5000 messages
  • Pioneer: 10000 messages

⚙️ Admin Controls

Admins have full control over leveling progression:

  • Grant or remove XP with simple commands.
  • Reset everyone’s progress when starting fresh.

🔔 Personal Settings

Each user can choose whether to see level-up notifications:

  • /enable-levelup → Enable level-up notifications
  • /disable-levelup → Disable level-up notifications

🎟️ Tickets & Verification

Unity provides a robust ticketing and verification system to keep your community organized and secure:

📝 Ticket System

  • Customizable Categories: Assign tickets to specific categories for better organization.
  • Support Roles: Designate staff roles that automatically gain access to tickets.
  • Ticket Transcripts: Automatically log and save full transcripts of closed tickets.
  • Multiple Ticket Types: Configure tickets for support, applications, reports, or other needs.
  • Close & Archive: Admins can quickly close tickets and generate a transcript for reference.

✅ Verification Tickets

  • Automated Verification: Require new users to open a verification ticket before joining the community fully.
  • Secure Role Assignment: Verified members are automatically given a designated role.
  • Streamlined Process: Reduces spam and ensures only legitimate users gain access.
  • Configurable Messaging: Fully customize the verification instructions and role assignment process.

📌 Reaction Roles System

Unity makes it simple for members to self-assign roles, helping your community stay organized and engaged:

  • Self-Assigned Roles: Members can pick their roles with a single reaction or button press.
  • Customizable Menus: Easily configure menus with multiple roles and unique labels.
  • Instant Role Updates: Roles are applied or removed immediately when a user reacts or unreacts.
  • Unlimited Configurations: Create as many reaction role menus as needed for your server.
  • Community Organization: Perfect for assigning interests, access levels, or event roles.

🛡️ Moderation

Unity provides powerful moderation tools to help you manage your community:

  • Clear Messages: Bulk delete unwanted messages.
  • ID Ban/Unban: Ban/unban users by their Discord ID.
  • Customizable Rules: Create and display fully tailored server rules with embeds.
  • AutoBan & Strikes: Catches banned words and enforces strikes automatically

⚠️ Severity Levels

  • Low: Issues a warning (two warnings escalate into one strike).
  • Medium: Directly issues one strike.
  • High: Directly issues two strikes.
  • Critical: Immediate ban from the server.

🔧 Server Setup & Configuration

Customize Unity to fit your server’s needs with easy configuration commands:

  • Configure moderation logs, join/leave logs, welcome channels, and more.
  • Set up reaction roles for community organization.
  • Track role membership counts in real time via voice channels.
  • Configure a complete ticket system with support roles and transcripts.

📊 Logging & Tracking Capabilities

Unity helps you stay on top of everything happening in your server with detailed logs and real-time tracking tools:

📝 Logging

  • Edit Logging: Capture when a message is edited, showing before and after.
  • Deleted Message Logging: Record when messages are deleted.
  • Banned Word Logging: Automatically log when a user triggers a banned word.
  • Level-Up Logging: Notify when users level up or down.
  • Welcome Message Logging: Send configurable welcome messages in a specific channel.
  • Join & Leave Logging: Track members joining, leaving, being kicked, or banned (with reason).
  • Ticket Transcription Logging: Save transcripts of ticket conversations.

📈 Tracking

  • UTC Time & Date: Configure a channel to always show the current UTC time and date.
  • Role Count Tracking: Display how many members hold a specific role in real-time.
    Example: If 10 users have the role Money, the channel name updates to show “Money: 10”.

🎲 Utility Commands

Unity also provides helpful tools for everyday use:

  • UTC Converter: Convert local times to UTC instantly.
  • Say Command: Have the bot repeat any message.
  • Giveaway Command: Run server giveaways by setting the prize, how many winners, and the duration.

📜 Commands List

🛡️ Moderation

  • /clear <amount> → Bulk delete messages (Admin only)
  • /id-ban <userID> → Ban a user by ID (Admin only)
  • /id-unban <userID> → Unban a user by ID (Admin only)
  • /strike <user> → Give a user a strike (Admin only)

🏆 Levels

  • /leaderboard → Show top 10 XP earners (Everyone)
  • /level-progress → Show personal progress (Everyone)
  • /add-messages <user> <amount> → Grant XP (Admin only)
  • /remove-messages <user> <amount> → Remove XP (Admin only)
  • /reset-all-messages → Reset all XP (Admin only)
  • /enable-levelup → Enable level-up notifications (Admin only)
  • /disable-levelup → Disable level-up notifications (Admin only)

🔧 Configuration & Tickets

  • /configure → Set channels for logs, welcomes, UTC, etc. (Admin only)
  • /configure-reaction-roles → Set up reaction role menus (Admin only)
  • /configure-role-count → Track role counts in voice channels (Admin only)
  • /configure-ticket-system → Configure ticket categories & roles (Admin only)
  • /verify-user <user> → Open a verification ticket (Admin only)
  • /unconfigure → Remove configuration (Admin only)

🎲 Utility

  • /get-utc <timezone> → Convert timezones to UTC (Everyone)
  • /say <text> → Repeat a message (Everyone)

📢 Send Commands

  • /send-giveaway-message → Send a giveaway embed (Admin only)
  • /send-roles-select → Send role selection menu (Admin only)
  • /send-rules → Send rules embed (Admin only)
  • /send-ticket-setup → Send ticket setup message (Admin only)
  • /send-verification → Send verification embed (Admin only)
